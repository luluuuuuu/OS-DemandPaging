# OS-DemandPaging
Second assignment of the OS course, SSE Tongji. Using demand paging algorithm.
## 项目目的
- 页面、页表、地址转换
- 页面置换过程
- 加深对请求调页系统的原理和实现过程的理解。

## 开发环境
- 操作系统平台:MAC OS
- 开发语言:html、css、js

## 置换算法
- 先进先出(FIFO)置换算法。
- 最久未使用(LRU)算法。
